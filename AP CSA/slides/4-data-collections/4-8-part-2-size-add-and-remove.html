ArrayList Methods

AP CSA reference methods

The ArrayList methods you need for the AP CSA exam include:

  • int size() returns the number of elements
  • boolean add(E obj) appends obj to the end and returns true
  • E remove(int index) removes the item at the index and shifts remaining items left

ArrayList Methods

Insert, access, and replace

  • void add(int index, E obj) inserts at an index and shifts current objects right
  • E get(int index) returns the item at the index
  • E set(int index, E obj) replaces the item at the index

E stands for the type of the element in the ArrayList.

size()

Number of items in the list

You can get the number of items in an ArrayList using its size() method.

The ArrayList starts out empty with a size of 0.

ArrayList<String> list = new ArrayList<String>();
System.out.println(list.size());

With arrays, use the length field. With an ArrayList, use the size() method.

add(obj)

Append to the end

You can add values to an ArrayList using add(obj).

This adds the object to the end of the list.

The source compares this to joining the end of the line to board a bus.

Autoboxing in ArrayLists

int to Integer

Primitive types like int and double are automatically converted to Integer and Double using autoboxing when added to an ArrayList.

When you pull an int value out of a list of Integers, that is called unboxing.

ArrayList<Integer> list = new ArrayList<Integer>();
list.add(new Integer(5)); // this will work in Java 7
list.add(5); // this will work in all Java versions

add(index, obj)

Insert and shift right

There are two different add methods in the ArrayList class.

add(obj) adds the passed object to the end of the list.

add(index, obj) adds the passed object at the passed index, but first moves existing values to higher indices to make room.

The indices for an ArrayList start at 0 and end at the number of elements - 1.

remove(index)

Remove and shift left

You can remove values from an ArrayList using remove(index).

It removes and returns the item at the given index.

This moves all the other items over in the underlying array and decreases the size of the ArrayList by 1.
