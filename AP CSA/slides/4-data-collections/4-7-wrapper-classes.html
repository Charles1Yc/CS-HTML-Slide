Wrapper Classes

Primitive values with object appearance

A wrapper class (包装类) wraps around a primitive data type and gives it an object appearance.

The wrapper classes are part of the java.lang package, which is imported by default into all Java programs.

The Integer class and Double class create objects from primitive types of int and double respectively.

Why Wrapper Classes

Objects, collections, and conversion

Sometimes you need to create a wrapped object for a primitive type so that you can:

  • give it to a method that expects an object
  • put it in a collection like an ArrayList
  • convert strings to primitive data types
  • convert primitive data types to strings

These conversions are useful when using the Scanner class with input.

Creating Integer and Double Objects

Older and newer Java syntax

In Java version 7, you can use a constructor like new Integer(2).

In Java 9 and later, you can use Integer i = 2; to create an object with the value 2 wrapped in it.

// in older versions of Java
Integer i = new Integer(2);
Double d = new Double(3.5);

// in newer versions of Java (9+)
Integer i = 2;
Double d = 3.5;

Integer and Double objects are immutable (不可变): once created, their values cannot be changed.

Integer Limits

Why those numbers?

The int type in Java can represent any whole number from -2147483648 to 2147483647.

Integers in Java are represented in 2’s complement binary and each integer gets 32 bits of space.

Integer.MIN_VALUE and Integer.MAX_VALUE are useful when you want to initialize a variable to the smallest possible value and then search a sequence of values for a larger value.

Autoboxing

Primitive to wrapper object

Autoboxing is the automatic conversion that the Java compiler makes between primitive types and their corresponding object wrapper classes.

This includes converting an int to an Integer and a double to a Double.

Integer i = 2;
Double d = 3.5;

Unboxing

Wrapper object to primitive

Unboxing is the automatic conversion from the wrapper class to the primitive type.

This includes converting an Integer to an int and a Double to a double.

Integer i = 2;  // autoboxing - wrap 2
int number = i; // unboxing - back to primitive type

Parsing Strings to Numbers

Integer.parseInt and Double.parseDouble

The Integer and Double classes have methods that convert strings to numbers.

They are often used with the Scanner class to convert input read as a String into an int or double.

These methods are listed on the Java Quick Reference Sheet.

  • static int parseInt(String s) returns the String argument as a signed int
  • static double parseDouble(String s) returns the String argument as a signed double

Coding Exercise

Run the code below to see useful parse methods in the Integer and Double wrapper classes.

public class Test2
{
    public static void main(String[] args)
    {
        Integer i = 2;
        Double d = 3.5;
        System.out.println(
                i.intValue()); // intValue() returns the primitive value
        System.out.println(d.doubleValue());

        String ageStr = "16";
        // Integer.parseInt and Double.parseDouble are often used to
        // convert an input string to a number so you can do math on it.
        System.out.println(
                "Age "
                        + ageStr
                        + " in 10 years is "
                        + (Integer.parseInt(ageStr) + 10));

    }
}
