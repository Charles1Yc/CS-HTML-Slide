get(index) and set(index, obj)

Access and replace

You can get the object at an index using:

obj = listName.get(index);

You can set the object at an index using:

listName.set(index, obj);

Both methods require that the index refer to an existing element of the list.

ArrayList Uses Methods

No bracket indexing

The index must be greater than or equal to 0 and less than the size() of the list.

ArrayLists use get and set methods instead of the index operator used with arrays:

array[index]

This is because ArrayList is a class with methods, not a built-in type with special language support like arrays.

Comparing Arrays and ArrayLists

When to use each

Use an array when:

  • you want to store several items of the same type
  • you know how many items will be in the array
  • the items will not change in order or number

Use an ArrayList when you do not know how many items you need or when you want to add or remove items while the program is running.

Create Arrays and ArrayLists

// arrays must specify a size!
int[] highScores = new int[5];
String[] names = new String[5];

// ArrayLists are empty to start with
ArrayList<Integer> highScoreList = new ArrayList<Integer>();
ArrayList<String> nameList = new ArrayList<String>();

Access and Change Comparison

length/size, access, modify

Operation array ArrayList
length/size array.length list.size()
Access value = array[index]; value = list.get(index);
Modify array[index] = value; list.set(index, value);

add and remove do not have simple equivalents in arrays because they change the number of elements and may shift positions.

ArrayList from Array

Not on the AP exam

Although it is not on the AP exam, you can convert an array to a List using the static method asList from the Arrays helper class:

Arrays.asList(arrayname)

ArrayList has a toString method that is automatically called to print the list in a nice format.

ArrayListFromArray Source Code

import java.util.*;

public class ArrayListFromArray
{
    public static void main(String[] args)
    {
        String[] names = {"Dakota", "Madison", "Brooklyn"};
        ArrayList<String> namesList = new ArrayList<String>(Arrays.asList(names));
        System.out.println(namesList);
    }
}

Expected output:

[Dakota, Madison, Brooklyn]
