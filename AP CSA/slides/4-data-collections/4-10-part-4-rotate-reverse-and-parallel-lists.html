Shift/Rotate an ArrayList

Rotate elements left or right

We can write code that shifts or rotates elements left or right in an ArrayList.

The source example rotates the elements in an ArrayList to the right by one position.

public static void rotateRight(ArrayList<Integer> nums)
{
    int last = nums.get(nums.size() - 1);
    for (int i = nums.size() - 1; i > 0; i--)
    {
        nums.set(i, nums.get(i - 1));
    } // end for loop
    nums.set(0, last);
} // end rotateRight method
