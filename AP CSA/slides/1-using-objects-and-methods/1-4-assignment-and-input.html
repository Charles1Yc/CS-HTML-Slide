Topic Intro

Variables change through assignment

Assignment statements (赋值语句) initialize or change the value stored in a variable using the assignment operator =.

Assignment statement diagram

Read score = points * 10 + 5; as score gets the value of points * 10 + 5, not as a math equation.

Assignment Statement

Variable left, expression right

An assignment statement always has a single variable on the left-hand side.

score = points * 10 + 5;
  • The right side is evaluated first.
  • The expression produces one value.
  • That value is stored in the variable on the left.

The variable on the left must be able to store the value’s type.

Assignment and Memory

Copy the value, not the variable itself

Set one variable’s value to a copy of another variable’s current value.

int x = 3;
int y = 2;
System.out.println(x);
System.out.println(y);
x = y;
System.out.println(x);
System.out.println(y);
y = 5;
System.out.println(x);
System.out.println(y);

After x = y;, changing y later does not change x.

Trace Task

Copy and update variables

Trace the code by hand before checking the answer.

int x = 0;
int y = 1;
int z = 2;
x = y;
y = y * 2;
z = 3;

What are the final values of x, y, and z?

Answer: x = 1, y = 2, z = 3.

Mixed-Up Code

Swap two values

Build the code that swaps the values in x and y.

Needed blocks:

int x = 3;
int y = 5;
int temp = 0;
temp = x;
x = y;
y = temp;

Distractor:

y = x;

The extra variable temp is needed because assignment overwrites the old value.

Data Types in Assignments

Compatible type required

Every variable must be assigned a value before it can be used in an expression. That value must come from a compatible data type.

  • A variable is initialized the first time it is assigned a value.
  • During execution, an expression evaluates to a single value.
  • The value of an expression has a type based on the operands and operators.

If an expression uses at least one double, the expression usually evaluates to double.

Reference Variables

null means no object reference

Reference types like String can be assigned a new object or null.

String str = null;
str = "new object";

null is a special literal that means the reference is not associated with any object.

  • null is not the same as an empty string.
  • "" is a real String object with zero characters.

Updating a Variable

Use the current value to make the next value

public class Test1
{
    public static void main(String[] args)
    {
        int score = 0;
        System.out.println(score);
        score = score + 1;
        System.out.println(score);
    }
}

Expected output:

0
1

The right side uses the previous value of score, then stores the new value back into score.

Update Experiment

Change the increment

Modify the update statement.

score = score + 2;

Predict:

  • What output changes?
  • What would happen with score = score + 10;?
  • Why is this still not a math equation?

This pattern prepares students for counters and loops.

Input with Variables

One algorithm, different user values

Variables are a powerful abstraction because the same algorithm can run with different input values.

Input can be:

  • tactile, such as clicking a button
  • audio, such as speech
  • visual, such as a webcam
  • text typed by the user

Java’s Scanner class is one way to obtain keyboard input.

Scanner Example

Store typed text in a variable

Keyboard input is useful in class, although it is not directly tested on the AP CSA exam.

import java.util.Scanner;

public class Main
{
    public static void main(String[] args)
    {
        System.out.println("Please type in a name in the input box below.");
        Scanner scan = new Scanner(System.in);
        String name = scan.nextLine();
        System.out.println("Hello " + name);
        scan.close();
    }
}

The same code works for any typed name.

Quick Check

Assignment with mixed numeric types

Consider the code segment.

int a = 5;
int b = a / 2;
double c = a / 2.0;
double d = 5 + a / b * c - 2;
System.out.println(d);

What is printed?

Answer: 8.0.

Reason: a / b uses integer division, then multiplication with c produces a double.

Classroom Check

A complete answer should…

  • explain assignment as storing the right-side value into the left-side variable
  • trace copied values through multiple assignment steps
  • use a temporary variable to swap two values safely
  • require compatible types before assigning or using a variable in an expression
  • recognize score = score + 1 as a valid update pattern
  • explain null as a reference value that means no object is associated
  • explain that Scanner stores keyboard input in variables, although keyboard input is not an AP exam target
