Topic Intro

Update and assign in one step

Compound assignment operators (复合赋值运算符) are shortcuts that do a math operation and assignment in one step.

x += 1;

means the same as:

x = x + 1;

Read the operator first, then the assignment: add, then store back.

Compound Operators

Written out versus shortcut

Written out Compound
x = x + 1 x += 1
x = x - 1 x -= 1
x = x * 2 x *= 2
x = x / 2 x /= 2
x = x % 2 x %= 2

The left-side variable receives the new result after the operation.

Mnemonic

Operation, then equal sign

If you are unsure whether to write += or =+, use the meaning.

  • The operation + is done first.
  • The result is assigned with =.
  • Therefore the operator is written +=.

The same pattern works for -=, *=, /=, and %=.

Increment and Decrement

Shortest forms for plus one and minus one

Because changing a variable by one is common, Java provides:

  • x++: increment (递增) x by 1
  • y--: decrement (递减) y by 1

Equivalent forms:

x++;
x += 1;
x = x + 1;

All three update x by one.

Prefix and Postfix Note

Know the rule, keep AP usage readable

Real Java code may use ++x or x++.

  • postfix: x++ uses the current value first, then changes it
  • prefix: ++x changes the value first, then uses it

AP CSA will not use the prefix form, and it will not use postfix in a context where the expression value matters.

Prefix and Postfix Example

Why AP avoids trick puzzles here

If x starts as 10:

System.out.println(x++);

prints 10, then x becomes 11.

If x starts as 10:

System.out.println(++x);

prints 11, and x is 11 afterward.

For this course, prefer clear updates on their own line.

Quick Check

Trace increment and compound addition

What are the final values?

int x = 0;
int y = 1;
int z = 2;
x--;
y++;
z += y;

Answer:

x = -1, y = 2, z = 4

Reason: subtract one from x, add one to y, then add the current y to z.

Quick Check

Trace mixed updates

What are the final values?

int x = 3;
int y = 5;
int z = 2;
x = z * 2;
y = y / 2;
z++;

Answer:

x = 4, y = 2, z = 3

Reason: y / 2 uses integer division because both operands are int.

Code Tracing

Dry-run code by hand

Code tracing (代码跟踪) means simulating a dry run through code line by line as if you are the computer.

Trace table example
