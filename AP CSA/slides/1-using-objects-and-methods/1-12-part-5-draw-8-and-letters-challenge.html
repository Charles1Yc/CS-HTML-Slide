Coding Exercise

activecode:: TurtleDrawEight

Can you make yertle draw the digital number 8, as 2 squares on top of each other?

The starter program creates a World and a Turtle.

Your code should add the movement and turn commands needed to draw the 8.

Starter Code

TurtleDrawEight

import java.awt.*;
import java.util.*;

public class TurtleDraw8
{
    public static void main(String[] args)
    {
        World habitat = new World(500, 500);
        Turtle yertle = new Turtle(habitat);
        // TODO: Make yertle draw an 8 with 2 squares.
        yertle.forward();

        habitat.show(true);
    }
}

Test Requirements

  • code contains .turn at least 5 times

The target drawing is two squares stacked on top of each other.

Groupwork Coding Challenge

Draw Letters

Working in pairs, use the area below to have your turtle draw simple block-style letters for your first or last name initials.

Use just straight lines:

  • no curves
  • no diagonals

It may help to act out the code pretending you are the turtle.

Turtle Direction Reminder

Facing north

Remember that which way you turn depends on which direction you are facing.

The turtle begins facing north, towards the top of the page.

Turtle Methods

Simple methods

Here are some simple turtle methods that you can use:

  • forward()
  • turnLeft()
  • turnRight()
  • backward()
  • penUp()
  • penDown()

Challenge Context

Simple methods first

You may notice that it is challenging to have your turtle draw with these simple methods.

In the next lessons, we will use more complex Turtle methods where you can indicate how many steps to take or what angle to turn.

That will make drawing a lot easier.

Classroom Check

A complete answer should include

  • use Turtle movement methods to draw a recognizable shape
  • use at least enough forward and turn calls for the Draw 8 task
  • create a Turtle object before calling its methods
  • use penUp() and penDown() when moving without drawing
  • remember that turns depend on the current turtle direction
  • keep habitat.show(true) at the end
