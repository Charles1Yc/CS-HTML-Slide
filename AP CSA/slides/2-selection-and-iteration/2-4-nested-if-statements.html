Topic Intro

Nested if statements

If statements can be nested inside other if statements.

Nested if statements consist of if, if-else, or if-else-if statements within if, if-else, or if-else-if statements.

The Boolean expression of the inner nested if statement is evaluated only if the Boolean expression of the outer if statement evaluates to true.

if (boolean expression)
{   // This nested if is executed if outer if is true
    if (boolean expression)
    {
        statement;
    }
}

Multiway Selection

else if

A single if/else statement allows us to select between 2 branches of code.

With nested if/else statements, we can pick between 3 or more branches of code.

A multi-way selection (if-else-if) is used when there are a series of expressions with different segments of code for each condition.

Multiway Execution

First true expression wins

Multi-way selection is performed such that no more than one segment of code is executed based on the first expression that evaluates to true.

If no expression evaluates to true and there is a trailing else statement, then the body of the else is executed.

Else If Pattern

Three-way choice

Just add else if for each possibility after the first if, and else before the last possibility.

// 3 way choice with else if
if (boolean expression)
{
   statement1;
}
else if (boolean expression)
{
   statement2;
}
else
{
   statement3;
}

Code Task

Code Window

activecode:: TryElseIf

public class TryElseIf
{
    public static void main(String[] args)
    {
        int x = 2;
        if (x < 0)
        {
            System.out.println("x is negative");
        }
        else if (x == 0)
        {
            System.out.println("x is 0");
        }
        else
        {
            System.out.println("x is positive");
        }
        System.out.println("after conditional");
    }
}

Quick Check

mchoice:: trace-nested-if

What does the following code print when x has been set to -5?

if (x < 0)
{
   System.out.println("x is negative");
}
else if (x == 0)
{
   System.out.println("x is zero");
}
else
{
   System.out.println("x is positive");
}

Options: x is negative, x is zero, x is positive.

Answer Reasoning

mchoice:: trace-nested-if

Correct answer: x is negative.

When x is equal to -5, the condition x < 0 is true. The later else-if and else branches do not execute.

The first condition is false and x is not equal to zero, so the else branch executes.

Quick Check

mchoice:: trace-nested-ifs3

What does the following code print when x has been set to .8?

if (x < .25)
{
    System.out.println("first quartile");
}
else if (x < .5)
{
    System.out.println("second quartile");
}
else if (x < .75)
{
    System.out.println("third quartile");
}
else
{
    System.out.println("fourth quartile");
}

Answer Reasoning

mchoice:: trace-nested-ifs3

Correct answer: fourth quartile.

The first three conditions are false for .8; the else branch prints whenever x is greater than or equal to .75.

Classroom Check

A complete answer should include

  • explain when the inner nested if is evaluated
  • describe multiway selection as first-true-branch execution
  • use else if branches for 3 or more choices
  • trace why later branches are skipped after a true condition
  • identify the final else as the branch for no true condition
