And, Or, and Not

Logical operators

What if you want two things to be true before the body of the conditional is executed?

Use && as a logical and to join two Boolean expressions. The body of the condition will only be executed if both are true.

Code Task

activecode:: compoundAnd

Textbook prompt: What if you want to go out and your parents say you can go out if you clean your room and do your homework?

Run the code below and try different values for cleanedRoom and didHomework and see what they have to be for it to print You can go out.

public class Test1
{
    public static void main(String[] args)
    {
        boolean cleanedRoom = true;
        boolean didHomework = false;
        if (cleanedRoom && didHomework)
        {
            System.out.println("You can go out");
        }
        else
        {
            System.out.println("No, you can't go out");
        }
    }
}

Logical Or

Inclusive or

What if it is okay if only one of two things is true?

Use || as a logical or to join two Boolean expressions. The body of the condition will be executed if one or both are true.

In programming, the or-operator is an inclusive-or: the whole expression is true if either one or the other or both conditions are true.

{
    public static void main(String[] args)
    {
        boolean walking = false;
        boolean carIsAvailable = false;
        if (walking || carIsAvailable)
        {
            System.out.println("You can go out");
        }
        else
        {
            System.out.println("No, you can't go out");
        }
    }
}

Numeric Range Checks

|| for error ends, && for in range

With numerical values, the or (||) operator is often used to check for error conditions on different ends of the number line.

The and (&&) operator is often used to see if a number is in a range.

Code Task

public class TestNum
{
    public static void main(String[] args)
    {
        int score = 10; // Try -10 and 110
        if (score < 0 || score > 100)
        {
            System.out.println("Score has an illegal value.");
        }
        if (score >= 0 && score <= 100)
        {
            System.out.println("Score is in the range 0-100");
        }
    }
}

Not Operator

! negates a Boolean value

The not (!) operator can be used to negate a Boolean value.

We’ve seen ! before in != (not equal). If you use ! in expressions with && and ||, be careful because the results are often the opposite of what you think it will be at first.

Operator Precedence

!, then &&, then ||

In Java, ! will be executed before &&, and && will be executed before ||, unless there are parentheses.

Anything inside parentheses is executed first.

Truth Table

P && Q

An expression involving logical operators like P && Q evaluates to a Boolean value, true or false.

P Q P && Q
true true true
false true false
true false ?
false false false

The result of P && Q is only true if both P and Q are true.

Truth Table Checkpoint

fillintheblank:: truthTableFill

Textbook prompt: The truth table above is missing one result. What is the result of P && Q when P=true and Q=false?

Correct response: false.

Reasoning: Both values must be true for && to return true.

Truth Table

P || Q

The result of P || Q is true if either P or Q is true. It is also true when both of them are true.

P Q P
true true true
false true ?
true false true
false false false

Truth Table Checkpoint

fillintheblank:: truthTableFill2

Textbook prompt: The truth table above is missing one result. What is the result of P || Q when P=false and Q=true?

Correct response: true.

Reasoning: Only one of the two has to be true with ||, so this will print true.
