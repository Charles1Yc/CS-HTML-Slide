# HTML Slide Style Direction Notes

## Pending Candidates

1. Textbook Annotation Style
   - Use textbook figures as the main visual.
   - Add numbered callouts, arrows, and short bilingual explanations around each key part.
   - Best for explaining camera parts, CCD, ADC, pixels, RGB values, and file transfer.

2. Technical Manual Style
   - Use clean header structure, restrained colours, technical diagrams, and compact explanation blocks.
   - Best for hardware/process lessons where students need precise terms and clear step-by-step logic.

4. Figure-First Classroom Style
   - Put one strong figure or process diagram at the centre of each slide.
   - Explain only the visible parts of the figure, with a short takeaway at the bottom.
   - Best for classroom teaching where students should understand the idea before reading details.
