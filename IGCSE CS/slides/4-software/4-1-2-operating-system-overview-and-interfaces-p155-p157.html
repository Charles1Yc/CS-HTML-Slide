Operating Systems

Why an operating system is needed

To enable computer systems to function correctly and allow users to communicate with computer systems, an operating system must be installed.

An operating system provides:

  • the environment in which applications can be run
  • a useable interface between humans and computer
  • a way to disguise the complexity of computer software and hardware

Operating Systems

Examples and storage

Common examples include Microsoft Windows, Apple Mac OS, Google Android and Apple IOS.

Most computers store the operating system on an HDD or SSD because operating systems tend to be very large programs.

Mobile phones and tablets store the operating system on a solid state device because they are too small to accommodate an HDD.

Human Computer Interface

CLI and GUI

The human computer interface (HCI) is in the form of a:

  • Command Line Interface (CLI)
  • Graphical User Interface (GUI)

A CLI requires typed instructions. A GUI uses pictures or symbols called icons.

Command Line Interface

Direct communication

A CLI requires commands to be typed in to choose options, open software, save or load a file.

The user must learn commands, and typing them each time is slow.

The advantage is that the user is in direct communication with the computer and is not restricted to pre-determined options.

Graphical User Interface

Icons and applications

A GUI allows the user to interact with a computer, MP3 player, gaming device or mobile phone using icons.

Selecting an icon can automatically load the application ready to be used.

WIMP Interfaces

Windows, icons, menu and pointing device

One common GUI technology is WIMP (windows icons menu and pointing device).

  • a mouse controls a cursor
  • icons are selected to open or run windows
  • each window contains an application
  • several windows can be open at the same time

Post-WIMP Interactions

Touchscreen actions

Devices such as mobile phones and tablets increasingly use touch screens and post-WIMP interactions.

Fingers are in contact with the screen, allowing actions such as:

  • pinching
  • rotating

GUI advantages GUI disadvantages
User does not need to learn commands Uses more memory than CLI

Pointing device or touch screen can launch applications Needs an operating system to operate

Interface Users

Who would use each interface?

CLI is used by a programmer, analyst or technician needing direct communication with a computer.

GUI is used by an end-user who runs software, plays games or stores and manipulates photographs.

Classroom Check

Explain OS purpose and interfaces

A complete answer should include:

  • an operating system provides an application environment and a human-computer interface
  • operating systems disguise software and hardware complexity
  • CLI uses typed commands and gives direct communication
  • GUI uses icons and is more user-friendly
  • WIMP uses windows, icons, menu and pointing device
  • post-WIMP uses touch actions such as pinching and rotating
