USB Overview

Universal connection standard

USB (Universal Serial Bus, 通用串行总线) is a form of serial data transmission. It is now the most common type of input/output port found on computers and has led to a standardisation method for the transfer of data between devices and a computer.

  • USB allows both half-duplex and full-duplex data transmission
  • It has become an industry standard, which means considerable support is available
  • Can support different data transmission rates (from 1.5 Mbps to 5 Gbps)

USB Cable Structure

Four-wired shielded cable

As Figure 2.12 shows, the USB cable consists of a four-wired shielded cable:

  • Red (+5 V) — power
  • Black (Ground) — power
  • White (Data +) — data transmission
  • Green (Data −) — data transmission

Two wires for power (red and black), two wires for data (white and green).

USB Device Detection

What happens when a device is plugged in

When a device is plugged into a computer using one of the USB ports:

  1. The computer automatically detects that a device is present (due to a small change in the voltage on the data signal wires in the USB cable)
  2. The device is automatically recognised, and the appropriate device driver software is loaded up so that computer and device can communicate effectively
  3. If a new device is detected, the computer will look for the device driver that matches the device; if not available, the user is prompted to download the appropriate driver software

USB-C

The new standard connector

A new type of USB connector, referred to as USB-C, is now becoming more common in laptops and tablets/phones:

  • 24-pin symmetrical connector — fits into a USB-C port either way round
  • Much smaller and thinner than older USB connectors
  • Offers 100 watt (20 volt) power connectivity — full-sized devices can now be charged
  • Can carry data at 10 gigabits per second (10 Gbps)
  • Can now support 4K video delivery
  • Backward compatible (to USB 2.0 and 3.0) provided a suitable adaptor is used
  • Expected to become the new industry standard (universal) format

Classroom Check

Understand USB and peripheral communication

A complete answer should include:

  • USB is a form of serial data transmission; allows both half-duplex and full-duplex
  • USB cable: four-wired shielded cable — two for power (red/black), two for data (white/green)
  • device plugged in → auto-detected via voltage change → device driver loaded
  • benefits: auto-detection, industry standard, no external power needed, error-free transmission, backward compatible
  • drawbacks: max 5 m cable length, older versions may not be supported, slower than Ethernet
  • USB-C: 24-pin symmetrical, 100 W power, 10 Gbps, 4K video, fits either way
